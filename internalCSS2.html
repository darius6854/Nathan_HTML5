HTML5/CSS3

วิธีการนำ CSS ไปใช้กับ HTML

1. Inline

เป็นการเขียน CSS เข้าไปในแท็ก HTML โดยใช้ style attribute

ข้อดี: ง่ายและสะดวกในการทำงาน เพราะเขียนลงไปใน Tag ได้เลย

ข้อเสีย: โค้ดไม่เป็นระเบียบ, ดูรก, ทำให้ไฟล์ใหญ่ขึ้น และแก้ไขยากหากใช้หลายจุด

<p style="color:#ffffff">ตัวหนังสือ</p>

2. Internal

เป็นการเขียน CSS ในไฟล์ HTML โดยอยู่ภายในแท็ก <style> ที่อยู่ใน <head>

ข้อดี: เขียนได้ง่าย และควบคุมสไตล์ได้ทั้งหน้าเพจ

ข้อเสีย: ไม่สามารถนำ CSS นี้ไปใช้ซ้ำในหน้าอื่นได้สะดวก

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>ชื่อเรื่องบทความ</title>
    <meta name="description" content="อธิบายบทความ">
    <meta name="keywords" content="คีย์เวิร์ดของบทความ">
    <style>
        body { background: #f2f2f2; padding: 10px; margin: 10px; }
        p { color: #ff0000; }
    </style>
</head>
<body>
    <p>ส่วนเนื้อหาของบทความ</p>
</body>
</html>
